# Backend API Development

A simple REST API built using **Node.js** and **Express.js**. This project demonstrates basic CRUD concepts by creating and retrieving user data using REST API endpoints.

## Features

- GET `/` - Welcome message
- GET `/users` - Get all users
- GET `/users/:id` - Get a user by ID
- POST `/users` - Add a new user
- Input validation
- Duplicate email validation

## Technologies Used

- Node.js
- Express.js
- Nodemon

## Installation

```bash
npm install
```

## Run the Server

```bash
npm run dev
```

## Server URL

```
http://localhost:5000
```

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Welcome message |
| GET | `/users` | Get all users |
| GET | `/users/:id` | Get a specific user by ID |
| POST | `/users` | Create a new user |
